# 🍔 Foodie Hunter Website

## 📌 Description
Foodie Hunter is a responsive web application that allows users to explore food items, restaurants, and offers through an attractive and interactive interface.

## 🚀 Features
- Responsive design
- Food menu display
- Attractive UI/UX
- Navigation bar
- Image gallery
- Interactive elements using JavaScript

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## ▶️ How to Run
1. Download the project
2. Open index.html in browser
